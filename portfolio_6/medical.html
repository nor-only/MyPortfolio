Common Symptoms

歯の症状

  • ✓ 歯が痛い
  • ✓ 歯がしみる
  • ✓ 歯が欠けた
  • ✓ 歯がグラグラする
  • ✓ 歯に穴があいた
  • ✓ 詰め物が取れた
  • ✓ 歯の色が気になる

歯ぐきの症状

  • ✓ 歯ぐきが腫れた
  • ✓ 歯ぐきから血が出る
  • ✓ 歯ぐきが痛い
  • ✓ 歯ぐきが下がった
  • ✓ 歯ぐきがかゆい
  • ✓ 歯と歯ぐきの境目が痛い
  • ✓ 歯ぐきの色が気になる

  • ✓ 口の中が痛い
  • ✓ 口が乾く
  • ✓ 口臭が気になる
  • ✓ 味がわからない
  • ✓ 口の中にできものがある
  • ✓ 舌が痛い
  • ✓ 顎が痛い・口が開かない
